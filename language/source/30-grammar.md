# Project Neral · 语法骨架

## 基本语序

- 默认语序：`SOV`
- 定语在前，中心词在后。
- 属格短语在前，领属名词在后。
- 关系词和格关系主要通过后置词处理。

## 词类轮廓

- 名词：不做复杂变格。
- 动词：主要通过独立体貌词和少量派生扩展。
- 形容词：可直接修饰名词，也可派生为状态名词和方式副词。
- 语法词：短小、稳定、拼写透明。

## 关键语法词

- `ge`：属格，表示“的”。
- `lo`：处所后置词，表示“在 / 于”。
- `de`：与向格，表示“向 / 给 / 为”。
- `se`：共格或工具格，表示“与 / 用”。
- `ko`：来源格，表示“从 / 自”。
- `na`：否定词，位于动词短语前。
- `pa`：过去。
- `ri`：进行。
- `vu`：意向 / 未来。
- `nu`：完成。
- `ke`：疑问句尾词。
- `si`：系词 / 存在词。

## 构词法

- 名词常见派生：
  - `-li`：关系形容词
  - `-to`：地点名词
  - `-ren`：相关人物
  - `-sa`：群体 / 场域
- 动词常见派生：
  - `-ma`：动作名词
  - `-ren`：施事者
  - `-te`：结果物
  - `-li`：可执行 / 与动作相关的形容词
- 形容词常见派生：
  - `-ma`：性质名词
  - `-ri`：变为某状态
  - `-se`：方式副词
  - `ku-`：反义或缺失

## 句法最低覆盖

V1 必须稳定表达：

- 陈述
- 否定
- 一般疑问
- 所有与归属
- 位置与移动
- 已然 / 进行 / 计划
- 比较
- 简单从句
